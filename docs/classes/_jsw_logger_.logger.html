Class Logger

Logger

module

Logger

constructor
since

1.0.0

classdesc

Logging module singleton which inherits the Winston Logger module. By default:

  1. Writes all the HANDLED exceptions under a log file in "logs/handledException.log"
  2. Writes in the console all warnings and erros

param

Enforcer internal object to avoid instanciating as "new Logger()"

param

Additional options

param

When set to true hides all logs (usefull when running tests)

param

Whether if throw an exception when logged trought the Logger#throw method
